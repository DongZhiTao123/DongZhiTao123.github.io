HEXO使用记录

1.安装node、npm->转换成cnpm

2.运行 cnpm install -g hexo-cli 命令行hexo -v验证安装

3.创建存放博客文件的文件夹

4.进入文件夹运行 hexo init 初始化hexo

5.hexo -s启动博客

6.创建markdown文件 放入博客目录下的source_posts 或者运行hexo new “My New Post”(文件名称) 命令创建一个markdown文件

7.退回博客目录运行hexo clean -> hexo g 重新生成

8.hexo -s 重新启动
