HEXO使用记录

1.安装node、npm->转换成cnpm

2.运行 cnpm install -g hexo-cli

可输入命令行hexo -v验证安装

3.创建存放博客文件的文件夹

4.进入文件夹运行 hexo init 初始化hexo

5.hexo -s启动博客

6.创建markdown文件 放入博客目录下的source_posts 或者运行hexo new “My New Post”(文件名称) 命令创建一个markdown文件

7.退回博客目录运行hexo clean -> hexo g 重新生成

8.hexo -s 重新启动

9.在github上new Repository

在命名是要注意名称要与github账号昵称相同

10.在博客目录下安装git部署插件 cnpm install –save hexo-deployer-git

11.设置博客目录下的_config.yml文件 拉到最后

最后一个branch默认为master可不设置

12.hexo d 部署到github上 期间会弹出github的登录窗口 登录即可

13.刷新刚刚new Repository后的页面会发现有文件在内

14.在浏览器中输入刚刚创建的仓库的网址:昵称+.github.io的一部分就可以访问到部署上的博客
